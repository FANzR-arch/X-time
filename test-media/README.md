# Test Media

This folder contains local media files for testing the extension media-library flow.

Generated:

- `sample-image.png`

Not generated automatically:

- `sample-video.mp4`

If you want to test video upload, put a small MP4 here named `sample-video.mp4`, or edit the Markdown `media:` field to match your own video filename. The extension test build limits each media file to 25MB.
